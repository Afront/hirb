Module Hirb::Console

  1. lib/hirb/console.rb

This class is meant to be extended to provide methods for use in a console/irb shell. For example: 

irb>> extend Hirb::Console
irb>> view 'some string', :class=>Some::String::Formatter
irb>> table [[:row1], [:row2]]

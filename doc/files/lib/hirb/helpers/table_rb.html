Base Table class from which other table classes inherit. By default, a table is constrained to a default width but this can be adjusted via options as well as Hirb:Helpers::Table.max_width. Rows can be an array of arrays or an array of hashes.

An array of arrays ie [[1,2], [2,3]], would render: 

+---+---+
| 0 | 1 |
+---+---+
| 1 | 2 |
| 2 | 3 |
+---+---+

By default, the fields/columns are the numerical indices of the array.

An array of hashes ie [{:age=>10, :weight=>100}, {:age=>80, :weight=>500}], would render: 

+-----+--------+
| age | weight |
+-----+--------+
| 10  | 100    |
| 80  | 500    |
+-----+--------+

By default, the fields/columns are the keys of the first hash.
